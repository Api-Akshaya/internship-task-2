# Internship Task-2: Build Your Personal Cybersecurity Lab

## Overview

This repository contains the implementation and evidence for **Task-2: Build Your Personal Cybersecurity Lab**.
The objective of this task is to create a safe and isolated environment for practicing basic cybersecurity concepts.

---

## Objective

* Set up a Kali Linux attacker machine
* Deploy a vulnerable web application
* Ensure lab isolation for safe practice
* Validate basic security tools

---

## Lab Environment

* **Host OS:** Windows
* **Virtualization Tool:** VirtualBox
* **Attacker Machine:** Kali Linux
* **Target Application:** OWASP Juice Shop (Docker-based)

---

## Tools Used

* Kali Linux
* VirtualBox
* Docker
* OWASP Juice Shop
* Nmap
* Burp Suite

---

## Implementation Summary

1. Kali Linux virtual machine was launched successfully.
2. Docker was installed and enabled.
3. OWASP Juice Shop was deployed locally.
4. Network configuration was verified.
5. Basic scanning and proxy validation were performed.

---

## Evidence

All evidence related to this task is available in the **evidence/** folder, including:

## Result

The cybersecurity lab was successfully set up in a controlled and isolated environment, meeting all the requirements of Task-2.

---

## Conclusion

This lab provides a foundational setup for future cybersecurity learning activities such as vulnerability assessment and web application security testing.
